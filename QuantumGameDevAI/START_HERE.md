## üîß Troubleshooting

### "API key not found"
Run this in terminal:
```bash
echo $OPENAI_API_KEY
```
If empty, set it:
```bash
export OPENAI_API_KEY='your-key-here'
```

### "Module not found"
Activate the virtual environment:
```bash
source venv/bin/activate
```

### Rate Limit Errors (Gemini)
Wait 60 seconds between calls, or use only ChatGPT
